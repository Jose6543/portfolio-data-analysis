## 01 · Data Cleaning Pipeline

Proceso de limpieza y normalización de datos crudos antes de cargarlos a Power BI.

**¿Qué hace?**
- Elimina duplicados y registros vacíos
- Estandariza formatos de fechas y texto
- Valida integridad referencial entre tablas
- Genera un dataset limpio listo para Power BI

**Stack:** SQL Server / PostgreSQL · Excel · Power BI
